**What this is about**

___

The core objective of this project was to learn how to design, build, and present an interactive chatbot using AWS Lex, an Amazon Web Services tool for creating conversational interfaces. The project was structured in two main parts:

**Part 1 Focus: Fundamentals**

    Goal: To understand the fundamental concepts of AWS Lex chatbot creation, including the relationship between intents and utterances.

    Outcome Target: To create a functional, simple chatbot and demonstrate knowledge of the platform's standard user interaction flow.

**Part 2 Focus: Advanced Features and Delivery**
    
    Goal: To build a more complex application (a knowledge quiz) within Lex.

    Outcome Target: To develop strong communication skills by presenting the project and explaining its technical functionalities to stakeholders.
___

**What I learned**

___

The project provided me with practical skills in conversational AI development.

**Part 1:**

- **Technical Setup**: I was able to set up and configure a chatbot using AWS Lex.
- **Design & Logic**:	I successfully created intent-driven responses aligned with user utterances.
- **Deployment**:	I deployed a simple chatbot for user interaction.

**Part 2:**

- **Technical Setup**: I built a functional knowledge quiz in Amazon Lex, demonstrating state management.
- **Design & Logic**: I applied effective strategies for complex conversational design.
- **Professional Skills**: I developed and demonstrated effective communication and presentation skills by preparing and delivering a live demo and successfully explaining chatbot functionalities to stakeholders.
